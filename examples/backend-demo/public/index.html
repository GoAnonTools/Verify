GoAnon Verify backend demo

This is a relying-party demo. The backend generates a single-use challenge, the browser asks the GoAnon Verify extension for an age proof, and the backend rejects reused challenges.

Flow

  1. Get a single-use challenge from this backend.
  2. Ask the GoAnon Verify extension for an age proof.
  3. Send the proof back to this backend.
  4. Backend verifies challenge, audience, threshold, expiry, and replay state.
